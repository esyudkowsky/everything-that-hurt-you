# Art-style experiments — the two leads

Restyles of **Avram** and **Her** from the existing masters
(`claude-notes/refs/`), to compare other frontier image models + art styles
against the shipped style (clean anime lineart, muted cel shading, via
`google/gemini-3.1-flash-image`). Each was generated by passing the master as a
reference and asking only the art style to change (character design/face/colors
kept). Magenta bg is deliberate — it's the chroma key the sprite pipeline cuts out.

Generated 2026-07-03 by `claude-notes/tools/run_style_experiments.sh`
(driver: `claude-notes/tools/gen_style.py`, which adds a `--model` flag to `gen.py`).

| Folder | Model | Style |
|---|---|---|
| `painterly/` | `google/gemini-3-pro-image` (Nano Banana Pro) | Painterly semi-realistic fantasy — rich brushwork, cinematic light, novel-cover key art |
| `watercolor/` | `openai/gpt-5.4-image-2` | Hand-painted watercolor storybook — soft washes, paper texture, loose ink lines |
| `retro90s_anime/` | `openai/gpt-5-image` | Vintage 1990s cel / OVA anime — film grain, retro palette, hard cel shadows |
| `ink_graphic_novel/` | `google/gemini-3-pro-image` (Nano Banana Pro) | Western graphic-novel ink — heavy blacks, cross-hatching, high contrast |

Each folder has `avram.png` and `her.png`. Per-call logs are `*.log` next to the folders.

## Notes / caveats
- **retro90s_anime** (`openai/gpt-5-image`) ignored the 3:4 aspect request and
  returned a **square** image — Her's wolf ears are clipped at the top and Avram's
  eye color drifted grayish. The style itself is the strongest departure of the four.
  If pursued, generate with headroom or a wider frame and re-crop.
- Both OpenAI models render a slightly different magenta than #FF00FF; fine for a
  look-test, but the sprite chroma-key (`bgremove.py`) tuning assumes true bright
  magenta from the Gemini pipeline.
- Gemini-3-pro (painterly, ink) held character identity + the exact magenta best,
  and honored 3:4. It's the natural pick if you want to keep the current pipeline
  and only swap the style suffix (per ART-PIPELINE.md "If redoing in a different style").

## If you like one
The shipped style lives in the `STYLE` suffix inside the four `make_*_jobs.py`
generators + the two lead-master prompts. Swapping styles = regenerate the 2 masters
in the new style, QA, then re-run every batch `--force` (everything keys off the masters).
